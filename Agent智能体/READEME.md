# AI Agent

1. 通用的大模型信息不是实时更新的,所有的大模型一定要去落地要业务场景上
2. 纯知识库就用RAG
3. Agent -- 循环决策机制
    LLM大脑（思考）+工具（执行）+ 循环决策机制
    我想去北京旅游，推荐一下穿搭
    接收到后先去思考，推理用户想去做什么事情，
    1.先去思考发现要查天气 2.调用天气的api做反馈 3.又进行思考去推理穿搭
    需要逐步进行多思考，
    降低大模型幻觉--RAG 多模态+工具增强（Agent）

4. Agent核心组件
    Tool（工具） 单个功能模块，比如查天气，查汇率，每个工具专注做一件事
    AgentType（代理类型）决定Agent如何思考和推理，常见的有ReAct（反应式推理）
    LLM（大模型）Agent的大脑，负责理解和推理，常用模型如qwen-turbo
    initialize_agent 把大脑（LLM）和工具们组合起来，生成一个能跑的智能助手
